It will allow user to show "At a Glance" of dashboard to the front-end. It will display total stats on front-end using widget. 

== Description ==

Check the [DEMO](http://allfables.com/) on our site, we have used the widget provided by this plugin into the right sidebar section of our demo site.

I believe to show "At a Glance" totals on the front-end to let my readers aware of it, so we designed this plug-in for our website. Now, we want to share this very useful plugin with everyone, for free!

**Why should you choose this plugins?**

* Very simple to use.
* Allows your user to share their "At a Glance" numbers to their readers.
* Fast and helpful support.
* Totally free.

#### Usage

* After installation activate the plugin.
* After activation you need to use widget "At a Glance" to show numbers wherever you like if your theme supports widgets. 


== Installation ==

1.  Extract the zip file and drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin.
2.  Use "At a Glance" widget to show total numbers at a glance statistics on front-end.
